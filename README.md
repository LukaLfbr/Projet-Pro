# Projet-Pro

Ce projet a pour objectif de démontrer mes compétences à travers la mise en place d’une infrastructure d’entreprise virtualisée, sécurisée et réaliste.

Il s’inscrit dans une démarche concrète de simulation d’un environnement professionnel, en intégrant des technologies couramment utilisées en entreprise (virtualisation, Active Directory, réseau, sécurité, etc.).

Ce dépôt a été conçu comme un support de démonstration technique, permettant d’illustrer mes capacités à concevoir, déployer et sécuriser une infrastructure complète.

! Le projet est actuellement en cours de finalisation. !
Certaines parties, notamment le monitoring, seront ajoutées ultérieurement.

Un sommaire détaillé est disponible afin de faciliter la navigation entre les différentes sections du projet.
